# Answer Generation Prompt

This prompt is responsible for injecting the retrieved context and the user's query into the final LLM call. It acts in conjunction with the `system_prompt`.

## Current `answer_prompt` Template

```markdown
Below is a list of retrieved chunks from academic database research papers. 
Use ONLY this information to answer the user's question.

### Context Chunks:
{context_blocks}

### User Question:
{user_query}

### Formatting Requirements:
- Use Markdown formatting.
- Use bolding for core concepts.
- Provide inline citations `[Title, Year, p. Number]` for all claims.
- If comparing systems, use a Markdown table.

Answer:
```

## Context Block Construction
The `{context_blocks}` variable is dynamically generated by the `prompt_builder.py`. Each block MUST be formatted to clearly separate the metadata from the text, so the LLM knows how to cite it.

**Example Block:**
```text
---
Paper: The Log-Structured Merge-Tree (LSM-Tree)
Year: 1996
Page: 4
Chunk ID: chunk_892
Text: 
The LSM-tree uses an algorithm that defers and batches index changes, cascading the changes from a memory-based component through one or more disk components in an efficient manner reminiscent of merge sort.
---
```
